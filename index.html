PYTHON Word guessing game (With AVALYN PHAN!)

This is a word guessing game that Avalayn Phan and I made for a project in computer science. The program first asks you if you'd like to guess 3 letter or 5 letter words, and after asking, the user is prompted to guess a word that is the same number of letters as the word. The program then searches through the guessed word and tells you if it is in the word, if it is correct in the right spot, or if it is incorrect. This trains the user to make inferences based on limited information.
