Dice Machine

Ever wanted to roll a six sided die but feel like a nerd while doing so? Using motion sensors, I created the code for this dice roller machine that, upon shaking, will select a random number from 1-6 after a short animation. This is a project from intro to programming and robotics, a class I took in 2020 during online learning.
